# Pazar
Pazar is a Single Page E-Commerce web application designed by using Bootstrap, Css for Styling Front-end.Spring MVC Framework & Hiberate ORM for Middleware & Back-end of the process.And H2 Database. 

"Pazar" is the Roman-alphabet transliteration of the Greek word , which translates to “bazaar” in English. Coincidentally, “Pazar” looks/sounds like the word “bazaar”
